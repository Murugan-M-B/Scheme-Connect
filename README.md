## Scheme Connect

## Overview
Scheme Connect is a full-stack web application built using Java and Spring Boot that enables users to efficiently discover, filter, and manage government schemes through a structured and user-friendly platform.

---

## Tech Stack
- Java
- Spring Boot
- MySQL
- HTML
- CSS
- JavaScript

---

## Features
- Browse and search government schemes
- Filter schemes based on eligibility criteria
- User-friendly interface for easy navigation
- Admin panel to manage schemes (Add/Edit/Delete)
- Efficient data handling using SQL database

---

## Project Structure
- Backend → Spring Boot (REST APIs)
- Frontend → HTML, CSS, JavaScript
- Database → MySQL

---

## How to Run the Project
1. Clone the repository  
2. Open backend in IDE (IntelliJ / Eclipse)  
3. Configure database in `application.properties`  
4. Run the Spring Boot application  
5. Open frontend in browser
